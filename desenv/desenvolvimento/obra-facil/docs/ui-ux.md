# UI / UX

## Objetivo

Definir a experiência do usuário e a estrutura visual do ObraFácil.

---

# Conceitos

O sistema deve transmitir:

* Confiança
* Simplicidade
* Rapidez
* Profissionalismo

---

# Público

## Clientes

Pessoas procurando profissionais.

---

## Profissionais

Prestadores de serviço.

---

# Princípios

## Mobile First

Todas as telas devem ser pensadas inicialmente para celular.

---

## Simplicidade

Menor número possível de cliques.

---

## Clareza

Informações importantes sempre visíveis.
