# Critério Final

O MVP será considerado pronto quando:

* Um profissional conseguir se cadastrar.
* Um cliente conseguir localizar um profissional.
* Um cliente conseguir iniciar contato via WhatsApp.
* Um administrador conseguir gerenciar a plataforma.
