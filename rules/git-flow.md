# Git Flow

## Branch Principal

main

Produção.

---

## Branch de Desenvolvimento

develop

Integração das funcionalidades.

---

## Features

feature/nome-da-feature

Exemplos:

feature/authentication

feature/professional-profile

feature/search-professionals

---

## Correções

bugfix/nome-do-bug

---

## Emergências

hotfix/nome-do-hotfix

---

# Commits

Formato:

tipo: descrição

Exemplos:

feat: create professional profile

fix: correct login validation

refactor: improve search service

docs: update architecture

---

# Pull Requests

Todo PR deve conter:

* Objetivo
* Impacto
* Evidências de teste
* Checklist
