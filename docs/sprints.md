# Critério de Sucesso do MVP

O MVP será considerado validado quando:

* Profissionais conseguirem criar seus perfis
* Clientes conseguirem localizar profissionais
* Clientes conseguirem iniciar contato via WhatsApp
* Administração conseguir gerenciar a plataforma
