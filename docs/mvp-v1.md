# MVP V1 - ObraFácil

## Objetivo

Validar a proposta de valor do ObraFácil:

Conectar clientes a profissionais de construção, reformas e manutenção residencial através de uma plataforma simples e intuitiva.

---

# Meta do MVP

Permitir que:

1. Profissionais criem seus perfis
2. Clientes encontrem profissionais
3. Clientes entrem em contato via WhatsApp

Sem intermediação financeira.

---

# Escopo do MVP

## Autenticação

### Cliente

* Cadastro
* Login
* Logout

### Profissional

* Cadastro
* Login
* Logout

---

# Perfil Profissional

O profissional poderá:

* Inserir foto de perfil
* Inserir descrição
* Informar WhatsApp
* Informar cidade
* Selecionar categorias
* Adicionar fotos do portfólio

---

# Critérios de Conclusão

O MVP será considerado concluído quando:

* Cliente conseguir localizar profissionais
* Profissional conseguir criar perfil
* Cliente conseguir iniciar contato via WhatsApp
* Administração conseguir gerenciar dados básicos
