# User Stories - MVP

## Objetivo

Definir as funcionalidades do MVP através de histórias de usuário.

---

# Cliente

## US001 - Cadastro de Cliente

Como cliente

Quero criar uma conta

Para acessar a plataforma.

### Critérios de Aceite

* Nome obrigatório
* E-mail único
* Senha obrigatória
* Conta criada com sucesso

---

## US002 - Login

Como cliente

Quero realizar login

Para acessar minhas funcionalidades.

### Critérios de Aceite

* Validar credenciais
* Gerar sessão autenticada

---

## US003 - Buscar Profissionais

Como cliente

Quero buscar profissionais

Para encontrar alguém que execute o serviço desejado.

### Critérios de Aceite

* Filtrar por categoria
* Filtrar por cidade
* Exibir resultados

---

## US004 - Visualizar Perfil

Como cliente

Quero visualizar o perfil de um profissional

Para avaliar se atende minha necessidade.

### Critérios de Aceite

* Exibir descrição
* Exibir categorias
* Exibir portfólio
* Exibir WhatsApp

---

## US005 - Contatar Profissional

Como cliente

Quero entrar em contato via WhatsApp

Para solicitar informações ou orçamento.

### Critérios de Aceite

* Botão WhatsApp visível
* Abrir conversa automaticamente

---

# Profissional

## US006 - Cadastro de Profissional

Como profissional

Quero criar minha conta

Para divulgar meus serviços.

### Critérios de Aceite

* Dados obrigatórios
* Perfil criado com sucesso

---

## US007 - Atualizar Perfil

Como profissional

Quero editar meu perfil

Para manter meus dados atualizados.

### Critérios de Aceite

* Alterar descrição
* Alterar cidade
* Alterar WhatsApp

---

## US008 - Selecionar Categorias

Como profissional

Quero escolher minhas áreas de atuação

Para aparecer nas buscas corretas.

### Critérios de Aceite

* Seleção múltipla
* Persistência dos dados

---

## US009 - Gerenciar Portfólio

Como profissional

Quero adicionar fotos dos meus trabalhos

Para demonstrar minha experiência.

### Critérios de Aceite

* Upload de imagens
* Exclusão de imagens
* Limite conforme plano

---

# Administração

## US010 - Gerenciar Usuários

Como administrador

Quero visualizar usuários cadastrados

Para controlar a plataforma.

### Critérios de Aceite

* Listagem
* Busca
* Ativação/Inativação

---

## US011 - Gerenciar Profissionais

Como administrador

Quero gerenciar profissionais

Para garantir qualidade da plataforma.

### Critérios de Aceite

* Visualização
* Edição
* Bloqueio

---

## US012 - Gerenciar Categorias

Como administrador

Quero administrar categorias

Para manter a organização dos serviços.

### Critérios de Aceite

* Criar categoria
* Editar categoria
* Ativar/Inativar categoria
